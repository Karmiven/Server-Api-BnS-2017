Purpose:
This directory is intended for storing files required to update the game client. It contains patches, scripts, installation components, and auxiliary resources necessary for automatic or manual updates of the game's client-side components.

Contents may include:
- Version metadata file: bns-patch\db\server.db.xxx.cab
- Patch packages: bns-patch\patch\4-1.cab, 5-1.cab, etc.
- Game client version file: bns-patch\Version.ini in the following format:

  [Version]
  ProductVersion=1.0.72.180 v 0
